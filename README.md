## Sobre o Projeto

API REST desenvolvida em Java com Spring Boot, com o objetivo de simular um sistema de gerenciamento de entidades.

### Funcionalidades
- Cadastro, consulta, atualização e remoção de registros
- Validação de dados no backend
- Tratamento de erros para requisições inválidas
- Persistência de dados em banco relacional
- Organização do código em arquitetura em camadas

### Tecnologias utilizadas
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- Banco de dados relacional
- Git
